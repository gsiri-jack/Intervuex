# IntervueX – Interview Practice Platform
A simple and structured platform designed to help users practice interview questions effectively.

---

## Overview
IntervueX is a lightweight interview practice system that enables users to record responses, upload videos, and review their progress.  
The goal is to provide a clean and focused environment for practicing interviews without unnecessary complexity.  
The platform is currently under active development, with core features being improved iteratively.

---

## Current Features
- Record and upload video responses  
  Supports full interview recordings or per-question uploads.

- Basic question flow  
  Users can attempt a sequence of predefined questions.

- AWS S3 integration  
  Backend supports secure video uploads through pre-signed URLs.

- Spring Boot backend  
  Manages video uploads and request handling.

---

## Future Scope
- AI-driven analysis  
  Automated feedback, answer evaluation, sentiment analysis, and speaking-pattern insights.

- Advanced question engine  
  Dynamic question generation based on difficulty, domain, and user performance.

- Full production deployment on AWS  
  Deployment planned using EC2, load balancing, secure APIs, and scalable storage on S3.

---

## Tech Stack
- Backend: Java, Spring Boot, MySQL  
- Frontend: React  
- Cloud: AWS S3 for video storage
